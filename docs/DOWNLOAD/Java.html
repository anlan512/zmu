Introduce

I

     Java是一种通用的 编程语言，它是基于类的，面向对象的（虽然不是纯粹的面向对象语言，因为它包含原始类型），并且设计为具有as 尽可能少的实现依赖。它旨在让应用程序开发人员 编写一次，在任何地方运行，这意味着编译的 Java代码可以在支持Java的所有平台上运行，而无需重新编译。 Java应用程序通常被编译为字节码无论底层计算机体系结构如何，它都可以在任何Java虚拟机（JVM）上运行。该语法的Java的类似于ç和C ++，但它有较少的低级别比二者设施。
